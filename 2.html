What is GitHub?

GitHub is a web-based platform that uses Git for version control and adds collaboration features. It provides a graphical interface for managing Git repositories and offers tools for code review, project management, and team collaboration.

Repositories

Host your Git repositories on GitHub with features like issue tracking, wikis, and release management. Public repositories are free, while private repositories offer enhanced security.

Pull Requests

Propose changes to a repository and discuss them with collaborators. Pull requests enable code review, automated testing, and merge conflict resolution.

Issues

Track bugs, feature requests, and tasks. Use labels, assignees, and milestones to organize and prioritize work effectively.

GitHub Actions

Automate your workflow with CI/CD pipelines. Build, test, and deploy your code automatically when you push changes.

GitHub Pages

Host static websites directly from your GitHub repositories. Perfect for project documentation, portfolios, and simple web applications.

Organizations

Manage multiple repositories and users under one organization. Control access, set up teams, and maintain coding standards across projects.

Collaboration Tools

GitHub provides various tools to enhance collaboration, including code review comments, project boards, and integration with popular development tools and services.

Connecting Git to GitHub

To connect your local Git repository to GitHub, you need to add a remote origin. This allows you to push your local commits to GitHub and pull changes from collaborators.

Authentication Methods

  • Personal Access Tokens: Recommended for command line and VS Code
  • SSH Keys: More secure for frequent operations
  • GitHub CLI: Authenticate through command line interface

Setting Up Remote Origin

# Add GitHub repository as remote origin
git remote add origin https://github.com/username/repo-name.git

# Verify remote
git remote -v

# Push local commits to GitHub
git push -u origin main

VS Code GitHub Integration

VS Code has built-in GitHub integration. Sign in to GitHub from VS Code to enable features like:

  • Publishing repositories directly to GitHub
  • Creating pull requests from VS Code
  • Managing issues and comments
  • Collaborating with team members

Publishing to GitHub from VS Code

  1. Open Source Control panel
  2. Click "Publish to GitHub"
  3. Choose public or private repository
  4. Sign in to GitHub if prompted
  5. Repository is created and pushed automatically

🔐 Security Best Practice

Always use Personal Access Tokens instead of passwords for GitHub authentication. Generate tokens from GitHub Settings > Developer settings > Personal access tokens.

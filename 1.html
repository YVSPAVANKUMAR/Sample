What is Git?

Git is a free and open-source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. It tracks changes in files and coordinates work among multiple people.

Basic Git Commands

  • git init: Initialize a new Git repository
  • git add: Add files to the staging area
  • git commit: Record changes to the repository
  • git status: Check the status of your repository
  • git log: View commit history

Example Workflow

git init
git add README.md
git commit -m "Initial commit"
git status
git log --oneline

Key Concepts

Understanding repositories, commits, branches, and merges is crucial for effective version control. A repository contains all of your project's files and each file's revision history.

Git Integration in Visual Studio Code

VS Code provides excellent Git integration out of the box. The Source Control panel allows you to perform all Git operations without leaving your editor.

Committing Changes in VS Code

  • Click the Source Control icon in the Activity Bar
  • Stage files by clicking the + icon next to changed files
  • Enter a commit message in the text box
  • Click the checkmark icon to commit

Pushing and Pulling in VS Code

# To push changes:
# Click the "..." menu in Source Control panel
# Select "Push"

# To pull changes:
# Click the "..." menu in Source Control panel
# Select "Pull"

Alternatively, use the status bar indicators at the bottom of VS Code for quick push/pull operations.

VS Code Git Features

  • Diff View: See changes side-by-side
  • Branch Management: Create and switch branches easily
  • Merge Conflicts: Resolve conflicts with built-in tools
  • Git History: View commit timeline and blame information

💡 Pro Tip

Use keyboard shortcuts in VS Code for faster Git operations:

  • Ctrl+Shift+G - Open Source Control
  • Ctrl+Enter - Commit with message
  • Ctrl+Shift+P - Command Palette for Git commands
